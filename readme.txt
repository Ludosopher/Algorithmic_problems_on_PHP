	Здесь представлены решения задач по курсу "Алгоритмы и структуры данных на PHP".

	В директории Optimization_of_large_calculations:
	- в файле Sum.php: Реализована функция a+b, где каждое из чисел а и b имеет не менее 1000 разрядов. Числа хранятся в файле chisla.txt
на двух строчках. Ответ вписывается на третью строчку;
	- в файле TextIteration.php: Для натуральных чисел a и n вычисляется a^n. Входные числа имеют диапазон 1 <= a <= 9 и 1 <= n <= 5000. Ответ выводится в файл otvet.txt.

	В файле bracket_balance.php проверяется баланс скобок в выражении, игнорируя любые внуренние символы.

	В файле explorer.php делается аналог «Проводника» в Windows для директорий на сервере при помощи итераторов.

	В файле largPrimeNum.php определяется самый большой делитель числа, являющийся простым числом.

	В файле Missing_search.php дан массив из n элементов, начиная с 1. Каждый следующий элемент равен (предыдущий + 1). Но в массиве  гарантированно 1 число пропущено. Находится и выводится на экран пропущенное число.

	В файле Repeatable_search.php находится количество повторений в массиве. Сложность O(logn) не должна измениться. Массив длиной n может состоять только из одного значения, например, [4, 4, 4, 4, ...(n)..., 4].

	В файле squareSpiral.php метод, принимающий на вход размеры двумерного массива и выводящий массив в виде инкременированной цепочки чисел, идущих по спирали.
